# 投资报告默认模板

按下面结构输出；若用户给了更具体格式，以用户格式为准。

## 报告头

```markdown
投资报告（YYYY-MM-DD）
报告日期（Asia/Shanghai）：YYYY-MM-DD
数据时间（UTC）：YYYY-MM-DDTHH:mm:ss.sssZ
价格口径：以“最近收盘日/最近可得收盘价”为基准（若为盘中则会标注；本次按各指数最近收盘日口径解读）
数据源：Yahoo Finance / 用户提供数据 / 其他来源
覆盖标的：标的1，标的2，标的3...

今日结论先行：
- 市场主判断：
- 仓位建议：
- 核心风险：
```

## 全局总结模板

```markdown
综合交易策略建议

当前主线：

核心风险：
全局异常提醒：
未来观察点：

免责声明：以上内容仅供参考，不构成投资建议，投资有风险，决策需结合自身风险承受能力并谨慎评估。
```

## 写作提醒

- 默认基于摘要做中文转述，关键新闻再抓全文
- 给关键新闻打 1~2 个标签
- 给关键新闻补信源等级（高 / 中 / 低）
- 最终建议增加置信度（高 / 中 / 低）
- 建议要先给明确动作，再给理由和关注点
- 输出前检查“操作建议”和“仓位建议”不能明显冲突
